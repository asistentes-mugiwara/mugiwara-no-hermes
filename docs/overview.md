# Overview

## Purpose

`Mugiwara no Hermes` is the public window into a private Hermes-based multi-agent system.

Its goal is to explain the system clearly enough to be useful, without disclosing live operational detail.

## System idea

The Mugiwara model is built around:

- a primary coordinator (`Luffy`)
- real specialist agents with differentiated scopes
- a shared canonical vault
- layered memory routing
- explicit delegation contracts
- operational discipline around tooling, runtime and publication

## What this repository versiones

This repository does **not** version the internal system directly.
It versions the **public representation** of that system.

That means releases here track:

- documentation maturity
- architectural clarity
- safe public examples
- showcase quality
- public governance of the project
